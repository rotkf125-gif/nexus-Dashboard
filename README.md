# NEXUS Dashboard - Deep Ocean Edition

**v2.0.0** | Next.js 14 Portfolio Dashboard with Deep Ocean Theme

## Theme: Deep Ocean

Glassmorphism + Bioluminescence 컨셉의 심해 테마 대시보드

### Color Palette

| Role | Color | HEX |
|------|-------|-----|
| Surface | Deep Navy | `#020617` |
| Deep | Abyss | `#0c1929` |
| Mid | Twilight Zone | `#1e3a5f` |
| Bioluminescence | Cyan | `#22d3ee` |
| Profit | Aqua Glow | `#22d3ee` |
| Loss | Deep Coral | `#f43f5e` |
| Warning | Gold | `#fbbf24` |
| Text | Pearl White | `#f0f9ff` |

### Features

- Depth gradient background (surface to abyss)
- Bubble particle animations
- Bioluminescent glow effects on hover
- Glass cards with cyan-tinted borders
- Floating hover transitions

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS + Custom CSS Variables
- **State**: React Context + Hooks
- **Charts**: Chart.js + Recharts
- **Testing**: Vitest

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
npm start
```
